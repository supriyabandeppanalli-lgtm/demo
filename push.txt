Push changes from local to remote repository:
1)git init
2)vi filename(push.txt)
3)git status
4)git add .
5)git commit -m "commit message"
6)git status
7)git remote add origin <http-link>:-This command is used to connect (link) your local Git repository to 
a remote repository hosted on platforms like GitHub
*git remote → manages connections to remote repositories.

*add → means you are adding a new remote connection.

*origin → is just a name (a label) for the remote repository (by convention, it means the main one).

*<http-link> → is the HTTPS link of your remote repository
(for example:https://github.com/supriyabandeppanalli-lgtm/demo.git).
8)git push -u origin branch-name(master):-This command is used to upload (push) your local branch and its commits to the remote repository (like GitHub).
It also links (tracks) your local branch with the remote branch so that in the future you can simply use
and always give the name of the branch that you are currently on and want to upload (push) to the remote.
